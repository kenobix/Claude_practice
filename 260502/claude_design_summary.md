# Claude Design 要約

## 概要

Claude Design は、Anthropic Labs が 2026年4月17日 にリサーチプレビューとして公開した会話型ビジュアル制作ツール。テキストプロンプトだけでUI・スライド・LP・プロトタイプなどを生成できる。搭載モデルは **Claude Opus 4.7**（ビジョン対応）。

---

## 利用可能プラン

Pro / Max / Team / Enterprise（追加料金なし、Enterprise はデフォルト無効で管理者が有効化が必要）

---

## 主な機能

### 1. デザインシステムの自動継承
- コードリポジトリ・既存スライド・ロゴ・スクリーンショットを読み込み、ブランドの色・フォント・コンポーネントを自動抽出
- 設定後は全プロジェクトに自動適用され、アウトプットのトーン統一が保証される

### 2. 対話ベースの制作フロー
- 左：チャットパネル / 右：キャンバスの二分割レイアウト
- 指示に対してClaudeが確認質問（目的・読者・トーン等）を選択肢形式で提示してから生成

### 3. バリエーション一括生成
- 初回から複数のレイアウト案を並べて提示し、方向性を比較・選択できる

### 4. 2種類の編集方法
| 方法 | 用途 |
|------|------|
| チャット | 全体構造の変更・トーンの転換 |
| インラインコメント | 特定要素へのピンポイント修正 |

また Tweaks（スライダーによる余白・色・密度の微調整）と直接テキスト編集も利用可能。

### 5. バージョン管理
「今の状態を保存して別アプローチを試して」と指示すれば、現バージョンを保持したまま探索できる。

---

## 作成できる成果物

- スライド・ピッチデック
- ランディングページ・ワイヤーフレーム
- インタラクティブプロトタイプ（クリック可能・フォーム動作あり）
- アプリUI（モバイル・デスクトップ）
- SVGアイコン・イラスト
- HTMLアニメーション（動画風コンテンツ）
- ダッシュボード・内部ツールのモック

## エクスポート・共有

| 形式 | 用途 |
|------|------|
| PDF | 印刷・共有 |
| PPTX | PowerPoint で追加編集 |
| ZIP | ローカル保管 |
| HTML（単一ファイル） | そのままWeb公開 |
| Canva | Canvaで仕上げ編集 |
| Claude Code ハンドオフ | 実装への引き継ぎ |

共有リンクは「閲覧のみ / コメント / 編集」の権限選択が可能。

---

## Claude Code との連携（最大の差別化ポイント）

デザイン完成後に「Claude Codeに引き継ぐ」を選ぶと、コンポーネント・デザイントークン・コピー・インタラクション情報を含む実装バンドルが生成される。これをClaude Codeに渡すだけで実装フェーズに移行できる。

```
Claude Design → ハンドオフバンドル → Claude Code → 実装
    設計             パッケージ化            実装
```

## 業界への影響

- Figma株価がリリース当日 -7% 下落
- Anthropic CPO の Mike Krieger 氏（元Instagram共同創業者）がリリース前日にFigmaのボードメンバーを辞任

## Anthropicの戦略的位置付け

Claude Design は仕事OS構想の一角。

| プロダクト | 担う職能 |
|-----------|---------|
| Claude Code | エンジニア（実装） |
| Claude Cowork | リサーチャー・オペレーター |
| Claude Design | デザイナー・ディレクター |

---

## 現時点の制限事項

- インラインコメントが消える場合がある（回避策：チャットに貼り直す）
- コンパクトビューで保存エラーが出る場合がある（回避策：フルビューで再試行）
- 大規模リポジトリでラグが生じる場合がある（回避策：サブディレクトリのみリンク）
- リアルタイム共同編集（Figmaのようなマルチプレイヤー）は未対応
- 公開APIは現時点で未提供
- PPTXエクスポートで日本語が画像化される場合がある
