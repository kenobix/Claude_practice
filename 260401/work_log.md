# 作業ログ: Claudeコネクタの設定と動作確認（2026-04-01）

## 概要

Claude.aiのコネクタ機能を設定し、外部サービスとの連携が正常に動作することを確認した。
チャットに入力するだけで外部データの取得や操作ができるようになり、日常業務の自動化が大きく前進した。

---

## 実施内容

### 1. コネクタの設定

Claude.aiの設定画面から「コネクタ」を開き、以下の外部サービスを連携設定した。

| コネクタ | 種別 |
|----------|------|
| Google Drive | 標準 |
| GitHub連携 | 標準 |
| Gmail | 標準 |
| Google Calendar | 標準 |
| Notion | 標準 |
| TickTick | カスタム（`https://mcp.ticktick.com`） |
| Figma | カスタム |

> コネクタとは、ClaudeがGoogleサービスやGitHubなどの外部アプリやサービスのデータを参照・操作できるようにする機能。

---

### 2. 動作確認

設定後、各コネクタが実際にデータを読み込めるかチャット上でテストした。

**確認方法**: チャットに自然言語で指示を入力し、Claudeが外部サービスのデータを取得・返答できるかを確認。

**結果**: 問題なく動作。チャットへの入力だけで外部情報を取得・操作できることが確認できた。

---

### 3. 活用できることが確認できたこと

| カテゴリ | 具体例 |
|----------|--------|
| タスク管理 | TickTick / Notionのタスク確認・作成・更新 |
| スケジュール管理 | Google Calendarへの予定確認・追加 |
| メール対応 | Gmailの受信確認・下書き作成 |

これまで手動でアプリを開いて行っていた作業が、Claudeへのチャット入力だけで完結するようになった。

## コネクタの役割と関連性

### コネクタ一覧と役割

| コネクタ | 主な役割 | できること |
|----------|----------|------------|
| **Google Drive** | ファイルストレージ | ドキュメント・スプレッドシートの読み取り・要約・検索 |
| **Gmail** | メール | 受信メールの確認、返信文の生成、下書き作成 |
| **Google Calendar** | スケジュール管理 | 予定の確認・作成・変更 |
| **GitHub連携** | ソースコード管理 | リポジトリのファイル読み書き、コードレビュー支援 |
| **Notion** | ナレッジ・タスク管理 | ページ・データベースの読み取り・作成・更新 |
| **TickTick** | タスク管理 | ToDoの確認・追加・完了操作 |
| **Figma** | デザイン管理 | デザインファイルの参照・コンポーネント情報の取得 |

---

### コネクタの関連図

```
                        ┌─────────────┐
                        │   Claude    │  ← チャットで自然言語入力
                        └──────┬──────┘
                               │
          ┌────────────────────┼────────────────────┐
          │                    │                    │
   ┌──────▼──────┐     ┌───────▼───────┐    ┌──────▼──────┐
   │  情報・知識  │     │  タスク・予定  │    │  コード管理  │
   └──────┬──────┘     └───────┬───────┘    └──────┬──────┘
          │                    │                    │
   ┌──────▼──────┐     ┌───────▼───────┐    ┌──────▼──────┐
   │Google Drive │     │Google Calendar│    │   GitHub    │
   │   Notion   │     │    Notion     │    │   Figma     │
   │   Gmail    │     │   TickTick    │    └─────────────┘
   └─────────────┘     └───────────────┘
```

### カテゴリ別の使い分け

```
【情報・知識】
  Google Drive ──→ ドキュメント・資料の参照
  Gmail        ──→ メールの確認・返信
  Notion       ──→ ナレッジベースの活用

【タスク・スケジュール】
  Google Calendar ──→ 予定の管理
  TickTick        ──→ ToDoの管理
  Notion          ──→ プロジェクト管理

【コード・開発】
  GitHub ──→ コードの読み書き・レビュー
  Figma  ──→ デザインファイルの参照・仕様確認
```

> **ポイント**: NotionはタスクとナレッジのどちらにもなるためGoogleCalendarやTickTickと組み合わせて使うと情報が一元管理できる。FigmaはGitHubと合わせることでデザイン〜実装のフローをClaude上で一気通貫できる。
